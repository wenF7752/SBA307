SBA 307 - Wen Fang
Project Overview
SBA 307 is a web application designed for a company that includes several key pages such as Login, Register, and Home (Index) pages. The site provides an easy-to-navigate interface for users to register, log in, and view various content such as news and information. The layout is fully responsive and adapts to various screen sizes, making it user-friendly across different devices like desktops, tablets, and smartphones.

Features
Responsive Design: The site is fully responsive using CSS Flexbox and media queries, ensuring it looks great on both desktop and mobile devices.
Forms: Includes form elements like registration, login, and user state selection using radio buttons and dropdowns.
Hover Effects: Various elements have hover and focus effects to enhance the user experience.
Grid Layouts: The site uses Flexbox for layout organization, ensuring elements are well-aligned and spaced evenly.
Styling: The site uses a consistent color scheme (based on pastel colors) and modern UI elements to maintain a clean and professional appearance.
CSS Animations: Smooth hover effects on buttons and cards to enhance the visual experience.
Technical Specifications
HTML5: For structuring the web pages.
CSS3: For styling the web pages, including responsiveness, layouts, and hover effects.
Flexbox: Used for layout management to make the site responsive and adaptable to various screen sizes.
Media Queries: Used to make the design fully responsive across multiple devices.
