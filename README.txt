SBA 307 - Wen Fang
Project Overview
SBA 307 is a web application project for a software engineering course at Perscholas. It includes key pages such as Login, Register, and Home (Index) pages. The site provides an easy-to-navigate interface for users to register, log in, and view various content such as news and information. The layout is fully responsive and adapts to different screen sizes, making it user-friendly across desktops, tablets, and smartphones.

Features
- Responsive Design: The site is fully responsive using CSS Flexbox and media queries, ensuring it looks great on both desktop and mobile devices.
- Forms: Includes registration, login, and user state selection forms using radio buttons and dropdowns.
- Hover Effects: Enhances the user experience with hover and focus effects on various elements.
- Grid Layouts: Uses Flexbox for layout organization, ensuring elements are well-aligned and evenly spaced.
- Styling: Maintains a clean and professional appearance with a consistent color scheme based on pastel colors and modern UI elements.
- CSS Animations: Adds smooth hover effects on buttons and cards for a visually appealing experience.

Technical Specifications
- HTML5: Used for structuring the web pages.
- CSS3: Used for styling the web pages, including responsiveness, layouts, and hover effects.
- Flexbox: Utilized for layout management to make the site responsive and adaptable to different screen sizes.
- Media Queries: Implemented to ensure the design is fully responsive across multiple devices.
